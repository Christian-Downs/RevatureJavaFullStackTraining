Intro to CSS

We have pretty good understanding of HTML so far, but it doesn't look to good to the user. The browser applies default styling such as Strong, which makes text BOLD. But how do we make our content look nicer?

CSS can be included in an html page in 3 ways:

  1. Inline (with the style attribute. This is not preffered.
  2. Using internal stylesheets (using the style element in the head.
  3. Using external stylesheets
CSS rules are defined by key-vlaue pairs, separated by a colon ( : ).
